Transformer-Based Feature Fusion Approach for
Multimodal Visual Sentiment Recognition Using Tweets in the Wild

在一项最新研究中，研究人员开发了一种基于变换器架构的实时多模态视觉情感识别系统，旨在解析社交媒体中的情感表达。该系统创新性地结合了面部图像、文本图像以及不包含面部或文本的图像，通过三种不同的模型提取各自的情感特征，并将这些特征融合，以增强在线情感分析能力。实验中，研究者利用全新的“领域无关多媒体情感数据集”（DFMSD）进行训练，同时运用了一种新提出的阈值移动方法，以有效应对社交网络数据中常见的数据不平衡问题。结果表明，采用阈值移动方法后，情感学习表现提升了5-8%。而通过引入两阶段培训策略后，整体准确率进一步提高了约12%。该研究展示了结合多种图像类型及先进的特征提取方法在促进社交媒体情感理解方面的显著潜力。未来，研究团队计划探索其他类型的特征，以进一步提高情感分析的准确性，并希望能将这一模型简化，以便在移动设备上实时应用。
